============
  LIMES Py
============

LOV Py provides an interface to LOV portal.
it most useful for tasks involving finding RDF classes and properties for strings. 
Typical usage often looks like this::

    #!/usr/bin/env python

    from lovpy import LovInterface

    LovInterface.findClasses(string)
